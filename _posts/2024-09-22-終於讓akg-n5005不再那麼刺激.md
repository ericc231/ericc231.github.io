前陣子看到AKG N5005價格大跳水的資訊後，就一直對它念念不忘，苦在找不到可以試聽的地方，最後還是決定盲狙，整理ㄧ些之前在網路上常看到的問題，跟實際使用後的狀況：

|  |  |
| --- | --- |
| 缺點 | 實際入手後的感受 |
| 佩戴感覺不佳 | 音管的角度的確稍微奇怪了些，放入耳朵後是斜的，所以使用原來習慣大小的耳塞會產生縫隙，使用較大的耳塞 使用隨附的spinfit耳塞就沒有問題了，後來有換很多文章推崇的Final Type E耳塞試試，並沒有感覺到什麼特別明顯的變化 |
| 難推 | 我的播放設備是iPhone 13 Pro Max + 原廠小尾巴，感覺也沒有難推的問題 |
| 耳管容易斷裂 | 這個待觀察，目前外觀正常 |
| 聲音太刺激 | 這個就有十足的感受了，聲音很豐富沒錯，但除了播放古典音樂外，整個聲音讓人感覺不舒服，隨著使用了一段時間後，情況有好一點，但仍是明顯感覺得到刺激感 |

太刺激、太刺激、**聲音真的太刺腦了**

目前仍打算維持用iPhone跟iPad來播放，不打算多帶一個設備，所以也沒考慮DAP，要改善大概只能靠接DAC（小尾巴）、換線等方式，到公館的加煒電子試用店員推薦的LETSHUOER DT03、狂派 KP1、Rose Technics RZ550以及自己想測試的KA17，用原廠的3.5線，聽的歌曲來源是Apple Music的無損音質，至少都是16bits/44.1 kHz ALAC

|  |  |
| --- | --- |
| 設備 | 測試心得 |
| 狂派 KP1 Rose Technics RZ550 | 這兩個小尾巴沒有改善惱人的刺激感 |
| LETSHUOER DT03 | 一開始聽就有個奇怪的感覺，彷彿世界變寧靜了，是的，那股刺激感不見了，取而代之好像聽到的一切都乖乖立正站好的感覺，似乎所有的一切都被磨平了，然後低頻的部分也變得顯著了起來，我喜歡這個寧靜感，但隱約總有隔層紗的感覺． 這個小尾巴的增益也有分高低，但沒有很大的差別，另外它的聲音控制是連動手機，所以沒法很細緻的控制聲音大小，不是太大聲就是太小聲，這也是最後我放棄它的原因 |
| Fiio KA17 | 號稱最迷你的台式耳擴，使用起來真的很熱情，如果沒有額外供給電源，iPhone Lightning可能會因為阻擋大電流而不能使用 KA17同樣也把那股刺激感去掉了，但卻沒有失去N5005原來那個豐富生態的奔放感，低頻也被明顯的增強，連續聽了一個小時，確定沒有被轟得頭暈腦脹，就決定帶走它了 KA17的音量控制跟播放設備是分開的，而且可以分得很細緻，可以調整到自己想要的音量 |

KA17的耗電量跟發熱程度，真的不是一般可以比擬，如果有在外使用的需求，一定要多帶額外的電源，否則手機的電可能很快就被耗光．
